# 🛒 Supermarket Product Tracking Simulator

This project simulates **product demand and inventory tracking in a supermarket** using Python, SimPy, and scikit-learn.  
It models customers, shelves, and restocking, with **AI-driven demand forecasting** to predict product demand and optimize inventory flow.

---

## 🚀 Features
- **Discrete-event simulation** of supermarket operations (customers, shelves, restocking).
- **AI forecasting** using scikit-learn to predict product demand by category.
- **Inventory management** with automatic restocking to prevent stockouts.
- **Extensible design** for adding dashboards, reinforcement learning, or IoT integration.

---

## 🧰 Tech Stack
- **Python 3.9+**
- **SimPy** – for discrete-event simulation
- **scikit-learn** – for demand forecasting
- **NumPy** – for numerical operations
